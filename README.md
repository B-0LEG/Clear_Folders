# Clear_Folders

Описание программы

Назначение:
Программа «Очистка пустых папок» (<i>EmptyFolderCleaner</i>) помогает найти и безопасно удалить все пустые папки в выбранной директории. Полезна при наведении порядка в файловой системе: после распаковки архивов, удаления файлов «вручную» или синхронизации часто остаются пустые каталоги, которые захламляют структуру. Приложение позволяет быстро выявить такие папки и удалить их массово.

Как пользоваться:

Выбор папки для поиска
В верхней части окна укажите путь к корневой папке, внутри которой будет выполняться поиск. Можно вписать вручную или нажать кнопку «Обзор…» и выбрать каталог в диалоге.

Настройка поиска
Галочка «Проверять вложенные папки» — если включена, программа будет рекурсивно обходить все подкаталоги и искать пустые на любом уровне вложенности. Если выключена — проверит только саму целевую папку.

Запуск поиска
Нажмите «Найти пустые папки». Во время поиска можно:

Приостановить / Продолжить — временно остановить обход.

Остановить — окончательно прервать поиск; уже найденные пустые папки останутся в таблице.

В строке состояния видно, какая папка проверяется в данный момент. По завершении (или остановке) таблица заполнится найденными путями.

Работа с результатами
Все пустые папки отображаются в таблице: столбцы «№», «Путь к папке», «Размер» (всегда 0). Доступны действия:

Выделить все — выбрать сразу все строки.

Очистить список — убрать все результаты (папки не удаляются, только список).

Удалить выделенные — удалить с диска выбранные папки. Будет запрошено подтверждение.

Удаление
При удалении программа пытается командой os.rmdir удалить только действительно пустые папки. Если какая‑то папка не пуста (например, туда что‑то попало за время между поиском и удалением), она не удалится, и будет выдано сообщение о количестве ошибок.

Важные замечания:

По умолчанию папки, содержащие только скрытые файлы (имена начинаются с точки), считаются пустыми и попадут в список. Если это нежелательно, вручную исключайте такие каталоги или проверяйте перед удалением.

Операция удаления необратима — удалённые папки не попадают в корзину. Удостоверьтесь, что вы выбрали правильные директории.

Для работы с системными или требующими прав доступа папками могут потребоваться соответствующие полномочия.
